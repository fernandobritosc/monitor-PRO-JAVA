# Alias: create-pr (MonitorPRO)

Este nome de skill é mantido para compatibilidade com o `agent-loop`.

## 🔄 Redirecionamento
Sempre que invocado via `@create-pr`, siga o fluxo de trabalho e as convenções documentadas em [**`pr-writer`**](../pr-writer/SKILL.md).

## 🎯 Quando Usar
- Quando você (usuário) pedir explicitamente para "criar um PR" ou usar o termo legado.
- Para organizar o trabalho final em um Pull Request limpo e estruturado seguindo os padrões Sentry.

## ⚠️ Nota Técnica
Use a `pr-writer` como a skill canônica para criar e editar pull requests.
