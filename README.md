## 🚀 Sobre o Projeto

O **Monitor Pro** é uma aplicação de alta performance projetada para estudantes de concursos de elite. Ele combina inteligência artificial avançada (Gemini & Groq) com uma interface fluida para proporcionar uma experiência de estudo organizada, orientada a dados e altamente produtiva.

## ✨ Recursos Principais

- 🧠 **IA Multi-Engine**: Integração com Gemini 2.0/1.5 e Groq (Llama 3) para geração de conteúdos, resumos e análises.
- 🎙️ **Podcast Style TTS**: Geração de áudio e podcasts para revisões utilizando as capacidades mais recentes de Text-to-Speech.
- 📊 **Dashboards Interativos**: Visualização de desempenho com Recharts e estatísticas em tempo real.
- ✍️ **Editor Rich Text**: Experiência de edição premium com Tiptap para criação de materiais de estudo.
- 🗂️ **Banco de Questões**: Sistema robusto de gerenciamento e resolução de questões com suporte a múltiplos formatos.
- ⚡ **Performance Nativa**: Interface ultra-rápida construída com Vite e Framer Motion para animações fluidas.

## 🛠️ Stack Tecnológica

### Frontend
- **Framework**: React 18 + TypeScript
- **Estilização**: Tailwind CSS + Framer Motion (Animações)
- **Estado**: Zustand + TanStack Query
- **Gráficos**: Recharts
- **Editor**: Tiptap SDK

### Backend & Serviços
- **Banco de Dados & Auth**: Supabase (PostgreSQL)
- **Inteligência Artificial**: Google Generative AI (Gemini) + Groq Cloud API
- **Analytics**: PostHog + Sentry (Monitoramento de Erros)

## 💻 Como Rodar Localmente

### Pré-requisitos
- Node.js (v18+)
- NPM ou Yarn

### Configuração

1. **Clone o repositório e instale as dependências:**
   ```bash
   npm install
   ```

2. **Configure as variáveis de ambiente:**
   Crie ou edite o arquivo `.env.local` na raiz do projeto:
   ```env
   VITE_SUPABASE_URL=seu_url_supabase
   VITE_SUPABASE_ANON_KEY=sua_chave_anonima
   VITE_GOOGLE_API_KEY=sua_chave_gemini
   VITE_GROQ_API_KEY=sua_chave_groq
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
